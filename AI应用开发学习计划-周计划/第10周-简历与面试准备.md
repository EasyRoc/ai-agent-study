# 第 10 周：简历、面试与双栈故事线

> 约 **5～7 个学习日**。目标：**字面上** 与**口述上** 都能把「8 年 Java + AI 项目经历 + 近几周系统补全的 RAG/Agent/工程化」**串成一条线**；不替代真实求职，只保证**不浪费**你已有经历。

## 本周目标

- 更新简历：**一段** 写清第 9 周或你**真实**工作中「Java + Python + 模型 + 可观测/配额」的 **STAR** 摘要（项目名可写「个人学习/离职期 Demo」**诚实**标注）。
- 准备 **1 分钟** 自我介绍 + **3 分钟** 项目深讲 + **5 个** 常见追问答案。
- 整理 **20 道** 场景题「口径」（不必背稿，有 bullet 即可）。

## 按日计划

### 第 1 天：经历分层

- [ ] 列表：**企业经历** 与 **第9周 Demo** 分栏，避免混成「线上经验」**不实**。
- [ ] 每段经历 3 bullet：**业务结果**、**技术栈**、**你个人**动作（用动词起句）。

### 第 2 天：双栈价值叙事

- [ ] 写 150 字内：**为何** 同时用 Java 与 Python（建议口径：*Python 负责数据侧/算法验证与 RAG 生态，Java 负责对外服务、与现有微服务/运维体系统一*；按你**真实**分工改）。
- [ ] 预演回答：**如果团队只要一端**，你愿意主抓哪边、如何过渡（**诚实**＋ 学东西快）。

### 第 3 天：技术追问清单（自测）

- [ ] RAG 什么时候会「胡说」？你**第4～5周**怎么**缓解**（引用、混合检索、拒答、评测）。
- [ ] 流式与同步选型；**首包** 延迟与**总**时长在 SLA 上分别代表什么。
- [ ] 幂等、长任务、Kafka 在**大模型**场景和**传统订单**有什么**相同/不同**（可结合你**车联网/算力**经历）。
- [ ] **Function Calling 与**「prompt 里写 ReAct 文本」**在工程上**你更信哪一个？**为什么**（可结合第 6 周笔记）。
- [ ] **ReAct** 里如何防**死循环/费用爆炸**（`max_iter`、**工具白名单**、**同工具重复**检测）？你 Demo 里**设了**什么？
- [ ] **MCP** 相对「再写 10 个 REST」对你**这个**项目的**价值**是？**若**不接入，REST **能否** 完全替代、**差**什么？
- [ ] **Skill** 和 **一坨 tools 都丢给模型** 有什么**可维护/产品**差异？（参考 `SKILL.md`）
- [ ] **RAG** 和 **用户长期记忆** 有什么**数据层** 区别？你的 Demo 里 **L2 与 L3** 分了吗？**怎么** 防串租户？（见 [`记忆架构-索引`](./记忆架构-索引.md)）
- [ ] 对话**摘要（L1）** 和 **RAG 检索（L2）** 什么**时候** 用谁？**能** 同时进上下文吗、**先** 谁**后** 谁、**费 token** 怎么控？

### 第 4 天：场景题 20 道（骨架）

- [ ] 为每题写 **3 个 bullet 要点**（不写作文）。建议覆盖类：
  - 多模型**降级/切换**、**限流**、**缓存** 穿透、
  - **客户**要「**必须**有出处」你怎么改架构、
  - **Token** 超预算、**提示词** 泄露/注入、
  - **并**发扣配额与 Redis 锁、
  - **线**上模型**输出** 不符合 JSON 的**重试**策略。

### 第 5 天：项目演示排练

- [ ] 屏幕共享：3 分钟从 **README** 到 **一问一答**（可提前录好，面试时放或现场）。
- [ ] 准备「**失败案例**怎么讲」：1 个检索不准 + 1 个你怎么**迭代**的（有 eval 或笔记更好）。

### 第 6 天：行为面与空窗/离职

- [ ] 写 **诚实** 口径（以你**真实**情况为准，下列仅结构）：**离职/裁员**原因一句 + **本段** 空窗在学什么（**指本计划**第几周到第几周产出）。
- [ ] 期望薪资与城市：**可**写区间 + **灵活**因素（不替你做决策，**你自己**填数）。

### 第 7 天：总复盘

- [ ] 一页 `interview-cheat-sheet.md`（**仅**给自己看）：**公司名留空**、**项目数字**、**反写** 要问面试官的 2 个问题（团队技术栈/模型是否自研/上线节奏等）。

## 周产出物

- [ ] 新版简历 `pdf` 或 `md`（**你自己**定稿，本文档不代写全篇）。

## 完成标准

- 模拟面试：**朋友**或自己录音 **30 分钟**，听回放 **卡顿 ≤ 3 次** 长沉默（**目标**，非强制）。

## 与持续学习

第 10 周结束后，若入职：把公司技术栈**映射**到本计划各周，**补**你弱的那几块即可；**不必**从第1周重学。
